# Recipe Index Website

## Project Overview
This website serves as a recipe index, featuring an organized collection of links to different recipes. It's designed as a practice project for HTML fundamentals learned through *The Odin Project* curriculum.

## Purpose
The main goal of this project is to apply and reinforce my knowledge of HTML by building a functional, organized webpage. It includes fundamental HTML elements such as:
- Tags and structure
- Links
- Images
- Lists and other essential HTML elements

## Technologies Used
- **HTML**: For structuring the content of the website

## About the Project
As I work on this project, I’ll focus on solidifying my understanding of HTML structure and best practices. Each recipe link will direct users to external sources or individual pages, showcasing various ways to organize and link content.
